## Design System
- Primary Color: #DC143C (Crimson Red - Bold)
- Secondary Color: #000000 (Pure Black)
- Tertiary Color: #FFFFFF (Pure White)
- Accent Color: #F5F5F5 (Light Gray for backgrounds)
- Typography: International Typographic Style (sans-serif, clean)
- Grid System: Strict mathematical grid with dynamic layout
- Spacing: Generous whitespace following Swiss design principles
